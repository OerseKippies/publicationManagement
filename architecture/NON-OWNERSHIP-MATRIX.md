# pubM Non-Ownership Matrix

Status: Architecture Foundation

| Concept | Not Owned By pubM | Owning Boundary |
|---|---:|---|
| User, Role, Permission, AccessPolicy, ServiceAccount | Yes | identityManagement |
| Breed definitions, product definitions, canonical identifiers | Yes | masterDataManagement |
| Stock, movements, reservations | Yes | inventoryManagement |
| Orders, customers, invoices | Yes | salesManagement / relationshipManagement / financeManagement |
| Advertisement, pricing, business rules | Yes | adManagement |
| Communication routing, service mediation, contracts | Yes | communicationLayer |
| Direct marketplace integrations | Yes | Outside pubM MVP foundation |
| External API gateway ownership | Yes | communicationLayer / governance-approved owner |
