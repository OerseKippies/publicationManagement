# pubM API Versioning Strategy

Status: DRAFT_IN_MODULE

## Initial Version

Initial accepted API version will be:

```text
v1
```

## Draft Location

Module draft:

```text
public/api/publication-api-draft.yaml
```

## Canonical Location

After OK-Core acceptance, the canonical contract should be stored in OK-Core:

```text
api/publication-api-v1.yaml
```

## Breaking Changes

Breaking changes require a new major version.

## Non-Breaking Changes

Compatible fields and endpoints may remain in the same major version after OK-Core review.
