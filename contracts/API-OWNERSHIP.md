# pubM API Ownership

Status: DRAFT_IN_MODULE

## Module Owns

pubM owns the draft semantics for publication lifecycle endpoints and events.

## OK-Core Owns

OK-Core owns canonical API governance, review, acceptance and canonical publication of accepted contracts.

## communicationLayer Owns

communicationLayer owns routing, mediation and contract registry behavior.

## Boundary Rule

The pubM API may mutate only pubM-owned entities.
